구글 드라이브 실시간 연동 설정

Google Drive에 배당 포트폴리오를 실시간 동기화하여 PC와 스마트폰 간에 데이터를 자동으로 주고받습니다.

구글 Client ID 발급 방법 (무료, 2분 소요)
  1. Google Cloud Console에 로그인합니다.
  2. 프로젝트를 생성하고 'Google Drive API'를 사용 설정합니다.
  3. 'OAuth 동의 화면'에서 외부(External)로 구성하고, 범위에 .../auth/drive.appdata를 추가합니다.
  4. '사용자 인증 정보'에서 [사용자 인증 정보 만들기] ➔ [OAuth 클라이언트 ID] (웹 애플리케이션)를 선택합니다.
  5. 승인된 JavaScript 원본에 다음 주소들을 추가합니다:
    - http://localhost:5000
    - http://127.0.0.1:5000
  6. 발급된 클라이언트 ID를 복사하여 여기에 붙여넣고 저장하세요.
